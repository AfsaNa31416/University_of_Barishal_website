# University of Barishal — Landing Page

This is a simple and responsive **landing page** for the **University of Barishal**, created using **HTML5** and **CSS3**.

## Features
- Navigation bar  
- Hero section with welcome text  
- About, Departments, Admission, and Contact sections  
- Developer information with photo and useful links  

## Technologies Used
- HTML  
- CSS  

## Files
- `index.html` — main webpage  
- `assets/css/style.css` — stylesheet  
- `assets/images/` — images used in the site
